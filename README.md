# Alchemy (beta) :)

**Alchemy** is a terminal-based TUI (Text User Interface) multimedia conversion tool written in **Rust**. It allows you to navigate directories and perform **image, video, and audio conversions** directly from your terminal.

---

## Features

- 📁 Navigate directories and files using Vim-like keybindings (`j`, `k`, `h`, `l`).  
- 🖼 **Image conversions:** PNG ↔ JPG  
- 🎥 **Video conversions:** MP4 ↔ MKV, MP4 ↔ WEBM  
- 🎵 **Audio conversions:** MP3 ↔ WAV, MP3 ↔ M4A  
- ⚡ Popup interface to add custom directories  
- 🔹 TUI highlights selected items for clarity  
- 🔄 Switch between directory navigation and conversion menus

## Dependencies
1. Make sure you have **Rust** installed. [Install Rust](https://rustup.rs/) if needed.
2. [ffmpeg](https://ffmpeg.org/)
3. [ImageMagick](https://imagemagick.org/script/download.php#gsc.tab=0)

   
## Installation

1. Install Dependencies
2. Clone this repository:
3. Cargo build and run
